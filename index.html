    Printer-specific notes: where to find Z-offset / leveling
    Bambu Lab (X1/P1/A1)Handled mostly by automatic bed mesh + LiDAR first-layer scan. Manual trim: Device panel → "Z-offset" calibration in the printer's touchscreen, or Studio → Calibration → Flow/Z-offset. A1/A1 mini also have a manual live-Z nudge during first layer print.
    Prusa (MK3/MK4/MINI)Live Z-adjustment: touchscreen/LCD → Settings → "Live Adjust Z" during the first layer, or Calibration → First Layer Cal. Mesh bed leveling runs automatically before each print (MK3S+/MK4).
    Creality / Ender (3/K1 etc.)Manual bed screws at each corner (paper test) on Ender 3-class printers; CR-Touch/BLTouch on newer models auto-meshes but still needs a Z-offset set via LCD → Prepare → Z-offset (adjust in 0.01–0.05 mm steps). K1/K1 Max use auto strain-gauge leveling with an on-screen Z-offset fine-tune.
    Klipper (generic)Use PROBE_CALIBRATE then TESTZ Z=-0.02 (or similar) to dial in z_offset in printer.cfg, or run BED_MESH_CALIBRATE. Save with SAVE_CONFIG. Many configs expose a "Z Offset" slider in Mainsail/Fluidd during a test print.

    Still stuck?

    Double-check belt tension and that your extruder gear is actually gripping filament (no chewed-up filament, no slipping). If it's still not clicking, post a well-lit, straight-down photo of the first layer to r/FixMyPrint — clear lighting and a close crop get the fastest, most accurate answers.
